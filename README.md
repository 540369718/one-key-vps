# one-key-vps
###自用一键配置VPS代码
1. 更换内核
2. 更新当前软件
3. 安装SS
4. 配置SS
5. 启动SS


使用方法
==========
1.下载脚本:
```shell
 wget https://raw.githubusercontent.com/540369718/one-key-vps/readme-edits/my-tmp.sh
```
2.运行(**如果有需要使用自己的根证书请将私钥命名为ca.pem，将根证书命名为ca.cert.pem，放到脚本的相同目录下再运行该脚本**)：
```shell
chmod +x my-tmp.sh
bash my-tmp.sh
```
